# 赛博抽抽机 🎲📺

一台浮在桌面上的复古像素小电视，是个万能随机决定器：拧旋钮换台，可以求签、决定中午吃什么、或抽任何你自定义的选项。点一下唰唰唰滚动，定格出结果。待机时屏幕显示可爱的颜文字表情 `・ω・`。

## 功能

**抽选**
- **求签频道**：7 级（大吉 / 中吉 / 小吉 / 末吉 / 平 / 凶 / 小凶，加权概率），每次配一句签文
- **列表频道**：如「中午吃什么」，从自定义选项里等概率抽一个；可在设置里增删频道、编辑选项

**换台 / 换肤（电视右侧两个旋钮）**
- 上旋钮 = 换台（循环切换频道，屏幕顶部显示频道名，切台有雪花效果）
- 下旋钮 = 换肤（4 套预设主题：绿萤光 / 琥珀橙 / 赛博紫 / 冰蓝，整机配色实时切换）

**设置面板**（hover 右上角 ⚙ 打开）
- 新建 / 删除频道、编辑选项列表与结语、选主题
- 频道、主题、自定义选项都会**持久化**保存，下次打开记住

**外观与交互**
- 复古 CRT + 像素风：扫描线 / 暗角 / 玻璃高光 / 雪花噪点 / 定格闪光 + 8-bit 音效
- **透明 + 无边框 + 总是置顶**，像桌宠浮在桌面
- 滚轮缩放整机；上下拖动天线可伸缩；拖动机身移动窗口

## 两种运行方式

### 1) 浏览器快速预览（零安装，看效果）

直接用浏览器打开 `src/index.html` 即可预览画面与动画。
（注意：浏览器里没有「透明 / 置顶 / 桌宠」效果，那是桌面应用专属。）

### 2) 桌面应用（完整体验，需要 Node）

本机还没装 Node，先用 Homebrew 安装：

```bash
brew install node
```

然后在项目目录：

```bash
npm install        # 安装 Electron（首次约几十 MB）
npm start          # 启动桌面应用
```

启动后小电视会出现在主屏幕右下角。

- 点击屏幕 / 红色电源键 / 空格键 = 抽一签
- 拖动电视机身 = 移动窗口
- 鼠标移上去，右上角 ✕ = 关闭（或 `Cmd + Q`）

## 交互一览

| 操作 | 效果 |
|------|------|
| 点屏幕 / 空格 / 回车 | 抽一下；出结果后再点 = 再抽一次 |
| 点电源键（待机时）| 抽一下 |
| 点电源键（出结果后，灯变蓝时）| 复位回颜文字待机 |
| 拧上旋钮 | 换台（切换频道）|
| 拧下旋钮 | 换肤（切换主题）|
| hover → ⚙ | 打开设置（管理频道 / 选项 / 主题）|
| 在电视上滚动鼠标滚轮 | 整机缩放（上滚放大 / 下滚缩小）|
| 上下拖动天线 | 拉长 / 缩短天线（向上拉变长，窗口随之长高）|
| 拖动机身 | 移动窗口 |
| hover → ✕ / Cmd+Q | 退出 |

> 🥚 隐藏彩蛋：求签频道里，天线越长越「欧」，拉到最长时大吉概率 **+5%**。

## 项目结构

```
src/
  main.js        Electron 主进程：透明置顶窗口、缩放贴合、配置读写、设置窗口
  preload.js     安全桥接：退出 / 缩放 / 配置 / 打开设置
  data.js        共享数据：主题、求签签库、默认频道
  index.html     电视结构
  style.css      像素 CRT 外观、主题变量、动画
  renderer.js    频道/主题/抽选逻辑、滚动动画、天线、音效
  settings.html / settings.css / settings.js   设置面板
  icon.png       运行时 Dock 图标
赛博抽抽机.app   双击启动器（指向本项目）
```

配置文件保存在 `~/Library/Application Support/赛博抽抽机/config.json`。

---

想改默认主题 / 签库 / 默认频道，编辑 `src/data.js`。日常增删频道与选项，直接用应用内 ⚙ 设置面板即可。
